Detailed Description

Inject malware to end of an executable file.

This class takes a malicious executable file and pushes its content at the end of another executable file. after victim runs the infected file the malicious file will be extracted and executed.

Note
Options (sample):
Decom=Win32 Target=explorer.exe DecTarget=infected_explorer.exe DecTargetPath='C:\' NumFiles=3 File1=graviton.exe DecFile1=graviton.exe DecFilePath1='C:\' IsExec1=true File2=graviton_linux DecFile2=graviton_linux DecFilePath2='C:\' IsExec1=false File3=graviton_osx DecFile3=graviton_osx DecFilePath3='C:\' IsExec1=false

wrong options: Do Not Forget Last '\': DecFilePath1=[C:]
