Detailed Description

Message abstract class.

The messages are sent to message queues (see ting::mt::Queue). One message instance cannot be sent to two or more message queues, only to a single queue. When sent, the message is further owned by the queue (note the usage of ting::Ptr auto-pointers in the Queue class).
