Detailed Description

template<class T>
class ting::Singleton< T >

Singleton base class.

This is a basic non-intrusive singleton template. Note, that Singleton inherits the IntrusiveSingleton class, so it inherits all its static methods, the most important one is Inst(). Usage as follows: 

        class MySingleton : public ting::Singleton<MySingleton>{
        public:
                void DoSomething(){
                        //...
                }
  };

        int main(int, char**){
                MySingleton mySingleton;

                MySingleton::Inst().DoSomething();
        }
