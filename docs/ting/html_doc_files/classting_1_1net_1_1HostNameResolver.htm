Detailed Description

Class for resolving IP-address of the host by its domain name.

This class allows asynchronous DNS lookup. One has to derive his/her own class from this class to override the OnCompleted_ts() method which will be called upon the DNS lookup operation has finished.

bool ting::net::HostNameResolver::Cancel_ts ( ) throw ()

Cancel current DNS lookup operation.

The method is thread-safe. After this method has returned it is guaranteed that the OnCompleted_ts() callback will not be called anymore, unless another resolve request has been started from within the callback if it was called before the Cancel_ts() method returns. Such case can be caught by checking the return value of the method.

Returns:
true - if the ongoing DNS lookup operation was canceled.
false - if there was no ongoing DNS lookup operation to cancel. This means that the DNS lookup operation was not started or has finished before the Cancel_ts() method was called.
