ting::Inited<volatile bool, false> ting::mt::MsgThread::quitFlag [protected]

Flag indicating that the thread should exit.

This is a flag used to stop thread execution. The implementor of Thread::Run() method usually would want to use this flag as indicator of thread exit request. If this flag is set to true then the thread is requested to exit. The typical usage of the flag is as follows: 

 class MyThread : public ting::MsgThread{
     ...
     void MyThread::Run(){
         while(!this->quitFlag){
             //get and handle thread messages, etc.
             ...
         }
     }
     ...
 };
